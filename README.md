# 基于可视化积木编程的深度学习教学平台

## 1. 安装Nodejs

## 2. 克隆仓库

```
git clone --recursive https://github.com/sunyia123/bbvdle.git
```

## 3. 安装依赖

```
cd bbvdle
npm install
```

## 4. 启动项目

```
npm run start
```

## 5. 访问项目

```
http://localhost:3000
```

## 6. 项目结构
```
bbvdle/
├── （暂缺）blockly-vue3/     # 前端代码
├── server/                  # 后端代码
└── README.md                # 项目说明文件
```
